## Requirements 
Node and NPM

## Running

### Settings
You can change the minimum and maximum wait time in the `guido.js` file, passing a number of milliseconds.

### Execute

Install dependencies 

```bash
npm i
```

Run 

```bash
npm start
```

or

```bash
node guido.js
```
